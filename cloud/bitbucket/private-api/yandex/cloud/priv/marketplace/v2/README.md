###Naming Java Classes
*Example*: PMPRPLIS
####prefix:
P - Private api
####project:
M - Marketplace
####entity:
* PB - **P**u**B**lisher
* P - **P**roduct
* PR - **P**artne**R**
* PRP - **P**artne**RP**roduct

####subentity:
* V - **V**ersion
* PL - **P**roposa**L**

####Service prefix
* C - **C**onsole
* I - **I**nternal
* P - **P**artners

####Suffix:
S - Service
